# Case Profile App

A simple React + Express application to create and manage case profiles with messages and tasks.

## Setup & Run

```bash
npm install
npm run dev
```

The above command starts:
- Express API on [http://localhost:3001](http://localhost:3001)
- Vite dev server on [http://localhost:5173](http://localhost:5173)

Use the web interface to create cases, upload documents, and manage messages and tasks.
